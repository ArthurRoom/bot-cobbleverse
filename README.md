# 🐉 Tutupadinha Bot - Cobbleverse & Santuário de Kanto

Bot de automação e gerenciamento para servidores de Minecraft (Cobblemon) e comunidades de Pokémon.

## 🚀 Funcionalidades
- **Status do Servidor:** Monitoramento em tempo real do status do servidor de Minecraft.
- **Recorde de Players:** Sistema de persistência que registra o pico máximo de jogadores online.
- **Cargos por Reação:** Automação de entrada para o Santuário de Kanto.
- **Hospedagem:** Configurado para rodar 24/7 na Discloud.

## 🛠️ Tecnologias
- Python 3.10+
- Discord.py
- Mcstatus (API de servidores Minecraft)
- Python-dotenv (Segurança de variáveis de ambiente)
